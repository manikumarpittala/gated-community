# Gated Community Management System

This project is developed using Java Servlets and JSP to manage complaints in a gated community.

## About

This project follows the MVC architecture using Servlets, JSP, and Hibernate. It allows residents to register, log in, raise complaints, and track their complaint status. The admin can view all complaints and update their status.

## Features
- User Registration and Login
- Raise Complaint
- View Complaint Status
- Admin Login
- View All Complaints
- Update Complaint Status

## Technologies Used
- Java
- Servlets
- JSP
- Hibernate
- MySQL
- HTML
- CSS
- Apache Tomcat
